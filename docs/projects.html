Projects

This page highlights research projects, benchmarks, and software I have built or contributed to, grouped by research theme. For code, see GitHub.

Benchmarks and Evaluation

mSCoRe

LREC-COLING 2026 · Code

A multilingual and scalable benchmark for skill-based commonsense reasoning. Evaluates LLMs across 5 languages and 10 reasoning skills with dynamic complexity scaling (L0–L3) that progressively increases task difficulty. Contains 5,600 instances spanning general and social/cultural commonsense.

MedRGB

AAAI 2026 Workshop on AI for Scientific Research · Code

A benchmarking framework for medical retrieval-augmented generation systems. Evaluates across four practical test scenarios — Standard-RAG, Sufficiency, Integration, and Robustness — with varying proportions of signal and noise documents, assessing reliability beyond accuracy alone.

Retrieval-Augmented Generation

ESV-RAG

Research project · Code

Explore–Solve–Verify Reasoning-Augmented Generation with Monte Carlo Tree Search planning. Formulates multi-hop question answering as a planning problem where an agent selects optimal action sequences through PUCT-guided tree search. Achieves state-of-the-art results across multiple FlashRAG benchmark datasets.

Multilingual Learning and Transfer

ChatGPT Beyond English

Findings of EMNLP 2023 · Code

Large-scale zero-shot evaluation of ChatGPT (GPT-3.5-turbo) on 7 NLP tasks across 37 languages spanning high-, medium-, low-, and extremely low-resource settings. Compares ChatGPT against state-of-the-art supervised models to quantify the multilingual gap.

Multilingual Relational Transfer (ZSRL)

Findings of ACL 2023 · Code

Zero-shot cross-lingual transfer for information extraction with multiple source and target languages. Introduces principled language selection via linguistic distance metrics and graph-relational domain adaptation (GrDA), yielding +3.0–3.7 F1 over random source selection.

Language-Disentangle-Distillation

Research project · Code

A compact multilingual pre-trained language model that explicitly disentangles language-specific and language-agnostic representations via knowledge distillation from XLM-RoBERTa. Uses dual hidden-state streams with Feature-wise Linear Modulation (FiLM) to address the curse of multilinguality.

Multilingual Federated Distillation

Research project · Code

A federated learning framework for multilingual NLP addressing data heterogeneity (different languages) and latency heterogeneity (different hardware). Combines semi-asynchronous tier training, personalized tokenizers, and federated distillation to reduce per-round communication payload by over 90%.

Domain Adaptation for Information Extraction

DA4JIE

Findings of EMNLP 2022 · Code

Unsupervised domain adaptation for joint information extraction across four tasks simultaneously (entity mention, event trigger, relation, and event argument extraction). Proposes Instance-relational Domain Adaptation (IrDA) with a chain graph topology and Context-invariant Structure Learning (CiSL).

DAA — Domain-specific Adapters for Event Detection

Findings of ACL-IJCNLP 2021 · Code

Unsupervised domain adaptation for event detection using three lightweight bottleneck adapters per BERT layer (source-private, target-private, joint). Combines layer-wise adversarial alignment with Wasserstein sample selection and orthogonality constraints.

MSP-DA

COLING 2022 · Code

Unsupervised domain adaptation for text classification via meta self-paced learning. Integrates MAML-style meta-learning with a neural self-paced learning module that dynamically partitions labeled source data into easy and hard subsets to simulate virtual domain shift during training.

Event Detection with Gumbel-Softmax Context Selection

PAKDD 2020 · Code

Event detection model that learns to select the most relevant context words using the Straight-Through Gumbel-Softmax estimator, handling discrete word selection as differentiable hard attention during training. Avoids hand-crafted dependency rules through iterative context updates.

Other Research Projects

Discourse Classification

Research project · Code

Argumentative discourse parsing for student essays. Addresses the 512-token BERT constraint via a Hierarchical Document Encoder (HDE) that splits documents into overlapping segments, classifying discourse elements (Lead, Claim, Evidence, Counterclaim, Rebuttal, Concluding Statement) at the sentence level.

Software and Tools

agent-maker

Software project

A declarative framework for compiling LLM agents from high-level specifications — interactive interview, preset shortcuts, or JSON spec driven. Supports 7 providers (Claude, Gemini, GPT, Perplexity, OpenAI, OpenRouter, GitHub Copilot) across 3 transport layers (browser automation, HTTP API, CLI subprocess) with a unified async runtime and multi-agent monitoring.

Kendo Translation Platform

Software project

A collaborative Japanese–English translation platform for kendo instructional content. Features a MAC-RAG (Multi-Agent Collaborative RAG) AI pipeline with keigo detection, terminology database, multi-candidate generation, quality scoring, and real-time collaborative editing backed by Supabase.
